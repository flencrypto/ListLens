🎯 Core Product Surfaces
ListLens Studio (/studio)

Create new listing sessions with photo uploads
AI-powered item analysis via a layered agent pipeline
Editable listing draft output (title, description, bullet points, condition, pricing)
Export to Vinted (manual export format)
Export to eBay (draft/sandbox/API-ready payload)
Per-item history and saved listings
ListLens Guard (/guard)

Paste URL or upload screenshots for risk checks
AI risk report: risk level (low/medium/high/inconclusive), confidence score, red flags, missing evidence
Safe-language guardrails — never says "fake", "counterfeit", "scammer" (enforced via sanitiseSafeLanguage())
Seller question generation
Guard report history
Dashboard (/dashboard)

Overview of listings, guard checks, and usage
History (/history)

Browse past Studio listings and Guard reports

🤖 AI Pipeline
Layered AI architecture (not a single prompt): Capture → Quality → Category Router → Identity → Evidence → Specialist Lens → Comps → Pricing → Risk & Trust → Copy → Validator
RecordLens ranked version likelihoods with confidence percentages
Guard AI risk reports with structured JSON outputs (Zod-validated)
Studio AI listing generation with structured outputs
All AI outputs include confidence scores, warnings, and missing-evidence flags
AI job metadata stored: prompt version, model, schema version, output, cost estimate
